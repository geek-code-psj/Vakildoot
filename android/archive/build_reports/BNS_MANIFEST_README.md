## Severity Tiers

The manifest classifies each section into one of four severity tiers based on statutory punishment:

### **Tier 0: Non-Penal** (231 sections - 64.5%)
- **Definition:** Sections with no criminal punishment
- **Examples:** Definitions (Section 2), General Exceptions (Sections 3-69), Procedural provisions
- **Characteristics:** Definitional, procedural, or exempting provisions

### **Tier 1: Minor** (28 sections - 7.8%)
- **Definition:** Community Service, fine only, OR imprisonment ≤2 years
- **Examples:** Defamation (fine/short imprisonment), Petty theft
- **Characteristics:** Low-severity crimes with limited custodial sentences

### **Tier 2: Moderate** (98 sections - 27.4%)
- **Definition:** Imprisonment >2 and ≤7 years
- **Examples:** Culpable homicide not amounting to murder, Theft, Cheating
- **Characteristics:** Mid-range crimes with substantial imprisonment

### **Tier 3: Severe/Heinous** (1 section - 0.3%)
- **Definition:** Imprisonment >7 years, Life Imprisonment, or Death Penalty
- **Examples:** Waging war against the State (Section 147), Murder with aggravation
- **Characteristics:** Most serious crimes against the state or person

## JSON Schema

Each line in the JSONL file contains exactly one JSON object with this structure:

```json
{
  "id": "BNS_SEC_106",
  "section_number": "106",
  "title": "PUNISHMENT FOR CAUSING HURT BY ACT ENDANGERING LIFE OR PERSONAL SAFETY",
  "severity_tier": 1,
  "punishment_summary": "Imprisonment up to 5 years, or fine up to ₹10,000, or both",
  "fair_use_summary": "Causing hurt by act endangering life or personal safety with specific intent.",
  "official_reference_link": "https://www.indiacode.nic.in/handle/123456789/20488?view_type=search&section=106",
  "app_safety_disclaimers": {
    "copyright_notice": "The statutory text of the BNS is in the public domain. This summary is provided for educational and navigational purposes only. Users must refer to the Official Gazette of India or IndiaCode for authoritative text.",
    "ai_generation_warning": "WARNING: This metadata, summary, and severity classification was generated by Artificial Intelligence. It has not been manually verified by a human and may contain errors or hallucinations.",
    "legal_liability_shield": "NOT LEGAL ADVICE. The developers of this application assume no liability for actions taken based on this information. Always consult a registered legal practitioner for legal proceedings."
  }
}
```

### Field Descriptions

| Field | Type | Description |
|-------|------|-------------|
| `id` | string | Unique identifier: `BNS_SEC_{section_number}` |
| `section_number` | string | BNS section number (1-358) |
| `title` | string | Full section title from the statute |
| `severity_tier` | integer | Severity classification: 0, 1, 2, or 3 |
| `punishment_summary` | string | Statutory punishment quantum |
| `fair_use_summary` | string | Brief 15-20 word summary of the offense |
| `official_reference_link` | string | Link to official IndiaCode source |
| `app_safety_disclaimers` | object | Legal disclaimers (see below) |

### Disclaimer Fields

**`copyright_notice`**
- Clarifies that BNS statutory text is public domain
- States summaries are for educational purposes only
- Directs users to official sources for authoritative text

**`ai_generation_warning`**
- Transparency notice about AI generation
- No human verification claim
- Acknowledgment of potential errors

**`legal_liability_shield`**
- Clear statement that this is NOT legal advice
- Developers assume no liability
- Users must consult registered legal practitioners
